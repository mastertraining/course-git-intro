## Basic Git Commands

Added all files to staging area

```
git add .
```

Commit the staging area to the repository

```
git commit -m "Commit message"
```

---

## View the log

```
git log
```

or in condense version

```
git log --oneline
```
or

```
git log --oneline --graph
```

---

## Getting help

```
git --help
```

or

```
git <command> --help
```

For example:

```
git log --help
```

---

## Checkout

```
git checkout HEAD~2
```

or

```
git checkout HEAD~1 -- Content.md
```
